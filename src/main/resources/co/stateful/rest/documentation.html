Stateful Web Primitives

It is a collection of simple web primitives, accessible through web UI and RESTful API.

Authentication

In every HTTP request you should set two authentication headers X-Stateful-URN and X-Stateful-Token to the values you see on top of this page.

For example, this is how your HTTP request may look like:

GET /c/test/inc?value=0 HTTP/1.1
Host: www.stateful.co
X-Stateful-URN: urn:github:526301
X-Stateful-Token: B28F-38E4-B305-C3A3
